RESUMEN

Tras el transcurso de dos semanas y la realización de las actividades de manera objetiva, se ha logrado evidenciar la metodología utilizada para que el programa basado en el siete y medio funcione de manera correcta, gracias al trabajo en equipo y buena uso de las herramientas se cumplió con el objetivo.
Viendo que todos los integrantes se han destacado en materias exactas propuestas dentro del proyecto, se ha de tener en cuenta que para la creación o desarrollo del juego dentro de Python todos han aportado en este.

Marc Pino

Comenzando con el primer integrante del proyecto, se destaca su manejo eficiente de MySQL, una habilidad derivada de los conocimientos adquiridos en la materia Base de Datos. Asimismo, hizo uso de la plataforma Azure, aplicando conceptos clave de la materia Sistemas Informáticos.

Gracias a estas herramientas, el programa logra gestionar datos de manera efectiva, incluyendo funciones como almacenar, visualizar y eliminar información. Esto permite una administración dinámica de las partidas, ajustes en los modos de juego, registro de jugadores y otros cambios necesarios para el correcto funcionamiento del sistema.

Laura Toro

En el caso del segundo integrante del proyecto, se destaca el uso de HTML y CSS para el diseño y desarrollo de la página web que presenta el contenido y el resumen del proyecto. Este enfoque permitió estructurar y estilizar el sitio de manera efectiva, asegurando una experiencia visual y funcional adecuada para los usuarios.

Además, se incorporó un simulador de puntaje, desarrollado principalmente utilizando JavaScript como herramienta clave. Este lenguaje estándar resultó fundamental para agregar interactividad y dinamismo a la aplicación, permitiendo una simulación precisa y atractiva. En conjunto, el uso de estas tecnologías refleja un manejo sólido de los lenguajes base en la creación de aplicaciones web modernas, alineándose con los objetivos planteados en la materia de Lenguaje De Marcas.

Rubén Bellido

Finalmente, el tercer integrante se encargó de implementar de manera eficiente el intérprete de Python, una herramienta fundamental para el desarrollo del proyecto. Python desempeñó un papel crucial al albergar toda la lógica y funcionalidad del juego, siendo el núcleo del comportamiento dinámico y las operaciones clave del sistema.

Este lenguaje destaca por su versatilidad y accesibilidad, y su uso en el proyecto refleja su importancia como protagonista en la materia de Programación. Su correcta aplicación aseguró un desarrollo sólido y eficiente del juego.
